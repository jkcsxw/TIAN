FAQ

What is TIAN?

TIAN is a project that helps users install and configure AI assistants, MCP integrations, and skills on their own computers.

Who is TIAN for?

It is designed for non-technical users, business operators, professionals, and developers who want a faster setup path.

Is TIAN only for Windows?

No. Windows has the smoothest no-code experience, but the project also supports macOS and Linux.

Where can I download TIAN?

The main download is the Windows installer on the GitHub Releases page. Source and scripts are in the GitHub repository.

TIAN is open source. Explore the code, releases, and setup instructions on GitHub.
